<pre>POST: [/game/api/v1/blackmarket/buy/:typeuid(\\\\d+)/:amount(\\\\d+)](#)</pre>

#### Response
```json
{
    "mReturnCode": 0, // optional; omit if 0
    "mMessage": "optional",
    "mDataBags": {},

    // BlackmarketResponse
    "CommodityName": "?",
    "CommodityUID": "?",
    "AmountPaid": 0, // Buy Price
    "AmountBought": 0.0
}
```

For details on `mDataBags`, see [Serialized Player Data](../../playerdata/serializedplayerdata).
