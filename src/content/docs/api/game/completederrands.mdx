<pre>GET: [/game/api/v1/completederrands](#)</pre>

#### Response

**On Error:**
```json
{
    "mReturnCode": 1,
    "mMessage": "mMessage"
}
```

---

**On Success:**
```json
[
    {
        "CompletedErrandID": 0,
        "PlayerDataID": 0,
        "mErrandUID": "?",
        "mTimesCompleted": 0,
        "mLastUpdated": "?"
    }
]
```

> An array of errands. May contain multiple entries.
