#### Response
The `requestedbags` parameter specifies which data bags to include in the response.

The same bitflags value is also returned in the response JSON as the `mRequestedBags` field, indicating which bags were actually included. See [Data Bags](/game/databags) for details on available flags.

The response is composed of a single JSON object that may include one or more of the data bags described below.

```json
{
    "mRequestedBags": 0 // omit if supplying no data bags
}
```

---
**Player Data:**

```json
{
    "PlayerDataBag":
    {
        "PlayerDataID": 0,
        "AccountID": 0,
        "Name": "?",
        "Triad": "?", // 18K, ShingWo, SunOnYee
        "EnforcerXP": 0,
        "Face": 0,
        "LastUsedVehicle": "?"
    }
}
```

---
**Shield:**

This should not be included in responses for the `/opponent/:index` endpoint.

```json
{
    "ShieldBag": 0 // Shield Timer in seconds
}
```
