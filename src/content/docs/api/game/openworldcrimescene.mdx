#### Response

**On GET:**
```json
{
    "mReturnCode": 0, // optional; omit if 0
    "mMessage": "optional",

    // OpenWorldCrimeSceneResponse
    "CrimeScenes": // optional
    [
        {
            "OpenWorldCrimeSceneID": "?",
            "DeathX": 0,
            "DeathY": 0,
            "DeathZ": 0,
            "VictimPlayerDataIDString": "?",
            "Collected": 0 // boolean
        }
    ]
}
```

---

**On POST:**
```json
{
    "mReturnCode": 0, // optional; omit if 0
    "mMessage": "optional",
    "mDataBags": {}
}
```

For details on `mDataBags`, see [Serialized Player Data](../playerdata/serializedplayerdata).
